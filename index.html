Campus Event Platform - Project Overview

Problem Statement

On my campus, there are 19 active forums, each organizing various weekly or monthly events. However, there is no centralized platform where forum leads can publish event details. As a result, many students miss out because event information is scattered across WhatsApp groups and social media, which require manual forwarding and often fail to reach everyone.

Solution Overview

I propose a single platform that consolidates all campus events in one place. It offers a real-time event feed categorized by clubs and departments for easy browsing. The platform features a chatbot powered by Google’s Gemini API, which answers students’ questions about upcoming events and offers personalized suggestions. Smart notifications with summarized updates keep students informed without overwhelming them.

Technologies Used

  • IDX: For frontend development and user interface management.
  • Firebase Studio: For backend services including Firestore real-time database and cloud messaging.
  • Gemini API: To power the AI chatbot and generate personalized event summaries.
  • Firebase Cloud Messaging: To send smart, summarized notifications to users.
